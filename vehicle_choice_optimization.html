Vehicle Choice Optimization Project

Vehicle Choice Optimization

This project predicts fuel consumption and CO₂ emissions for vehicles in Canada using machine learning. It empowers consumers to make eco-conscious vehicle decisions using a real-time, interactive dashboard. The dataset spans 2015–2025 models and is sourced from the Government of Canada.

Technologies Used

  • Python (Pandas, Scikit-learn, XGBoost)
  • Jupyter Notebook
  • FastAPI for backend
  • HTML, CSS, JS for frontend
  • Chart.js for data visualization
  • Render for deployment

Steps Involved

  1. Data Collection and Cleaning
  2. Feature Engineering & EDA
  3. Model Training (Linear, Ridge, XGBoost)
  4. Hyperparameter Tuning & Outlier Removal
  5. Deployment via FastAPI

Key Insights

The XGBoost model was selected as the final model due to its ability to capture non-linear patterns and deliver highly accurate predictions. It achieved a test R² score of 0.9959, and had the lowest average MAE among all models. Fuel type, engine size, and fuel consumption were the most influential features in predicting CO₂ emissions.
