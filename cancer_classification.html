Breast Cancer Diagnosis Project

Breast Cancer Diagnosis Prediction

This project focuses on predicting whether a breast tumor is benign or malignant using machine learning models. The dataset used is the Wisconsin Diagnostic Breast Cancer dataset (WDBC) obtained from the UCI Machine Learning Repository.

Technologies Used

  • Python
  • Jupyter Notebook
  • Scikit-learn
  • Pandas, Matplotlib, Seaborn

Steps Involved

  1. Data Loading and Preprocessing
  2. Exploratory Data Analysis (EDA)
  3. Model Training (Logistic Regression, Random Forest, SVM)
  4. Model Evaluation and Comparison
  5. Feature Importance Analysis

Key Insights

All models achieved high accuracy (AUC > 0.98). SVM and Random Forest were the top performers. Features such as 'Radius Worst', 'Perimeter Worst', and 'Concavity Worst' were critical in predicting tumor malignancy.
