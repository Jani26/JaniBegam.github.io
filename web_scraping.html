Web Scraping Using Python

This project involved web scraping the Amazon Best Seller Books webpage using Python and comparing the performance of web scraping tools.

Technologies Used

  • Python
  • BeautifulSoup
  • Pandas
  • Matplotlib & Bokeh

Key Objectives

  1. Scrape book details such as Title, Author, Rating, and Price.
  2. Store data in a structured CSV format.
  3. Perform EDA (Exploratory Data Analysis) to handle missing data.
  4. Generate insights using visualizations.
  5. Compare performance of BeautifulSoup, lxml, and Regex.

Highlights

The scraper fetched details of the top 60 books, despite challenges with dynamic webpage structures. Visualizations provided insights into:

  • Top 25 highest-priced books
  • Top-rated books based on customer ratings
  • Performance comparison of scraping tools
