Mushroom Classification

This project involves building a classification model to predict whether a mushroom is edible or poisonous using Python and machine learning libraries. The dataset was obtained from UCI Machine Learning Repository.

Technologies Used

  • Python
  • Jupyter Notebook
  • Scikit-learn
  • Pandas, Matplotlib, Seaborn

Steps Involved

  1. Data Cleaning and Preprocessing
  2. Exploratory Data Analysis (EDA)
  3. Model Selection and Training
  4. Evaluation and Optimization

Key Insights

The model achieved an accuracy of 95%, highlighting the importance of specific mushroom features in determining edibility.
