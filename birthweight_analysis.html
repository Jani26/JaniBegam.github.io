Birthweight Impact Analysis

This project investigates the factors influencing newborn birthweight using statistical techniques such as: T-Tests, Linear Regression, and Multiple Regression models. The dataset includes 1174 observations and 7 predictors.

Technologies Used

  • R Programming
  • Data Visualization Tools
  • Statistical Modeling

Key Steps

  1. Descriptive Statistics
  2. Data Visualization
  3. T-Test for Birthweight
  4. Simple Linear Regression
  5. Multiple Linear Regression

Key Findings

- Smoking reduces birthweight by approximately 262 grams.
- Gestation, height, and weight positively impact birthweight.
- The model explains 25% of the variation in birthweight.
